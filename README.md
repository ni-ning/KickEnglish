##### 第002课 你真的会读/ɪ/吗？ 这个音至少一半的人都会读错，到底该怎么读？

/i/ 长音？
```
I am reading a book.
I'am glad to see you.
This is a book.
```
- 一般情况下的发音方式决定"长/短"，说出来/弹出来
- 破除 "元音成对"

一句话要领：`嘴角放松 轻轻弹出`

松元音
```
English
We use our produts in non-English languages.

sit
Like we sit in front of the screen...

sister  will /wɪ /

lick    him
fill    milk
```

- 发音方式定"长短"
- 嘴角放松 轻轻弹出
- "弹出来"的发音习惯
